Front-end ծրագրավորող

Ստեղծում եմ մաքուր, արագ և գրավիչ վեբ կայքեր։

Իմ մասին

Ես Արմեն Սարգսյանն եմ։

Սովորել եմ Full Learning-ում՝ որպես վեբ ծրագրավորող՝ մասնագիտանալով Front-end մշակման մեջ։

Տիրապետում եմ HTML, CSS և JavaScript, ինչպես նաև ծանոթ եմ Git-ի աշխատանքին։

Այս կայքը ստեղծել եմ որպես անձնական պորտֆոլիո՝ ապագայում ներկայացնելու իմ նախագծերը։ Ես հավատում եմ կայուն աճին, թիմային աշխատանքին և մաքուր կոդին։

Իմ հմտությունները

HTML5

Կառուցվածքային մաքուր նշագրում կայքի հիմքի համար։

CSS3

Պատասխանատու դիզայն և ճկուն դասավորություններ։

JavaScript

Դինամիկություն և ինտերակտիվություն կայքի համար։

Git

Տարբերակների վերահսկում և թիմային համագործակցություն։

Figma

Մակետների ուսումնասիրություն և դիզայնի հետ սերտ աշխատանք։

React (շուտով)

Ժամանակակից բարդ ինտերֆեյսերի կառուցում։

Կապ հաստատեք

Պատրաստ եմ համագործակցության, նախագծերի կամ գաղափարների քննարկման համար։
